ToolTune

          
Teaching language models when and how to use tools through reinforcement learning.

        


        

          

            Problem
            
Base language models cannot reliably invoke external tools. They hallucinate tool calls, use wrong parameters, or refuse to act when tools are needed. Fine-tuning on format alone (SFT) teaches structure but not judgment.

          

          

            Approach
            
ToolTune applies a two-stage training pipeline: SFT for format compliance, then GRPO (Group Relative Policy Optimization) to teach when to use tools, which tools to pick, and when to stop. The reward function scores tool accuracy, restraint, efficiency, and answer quality.

          

          

            Results
            
Task accuracy: 8% (base) to 60% (SFT) to 62% (GRPO). The real win is behavioral: GRPO models show restraint (not over-tooling), use targeted queries, and cite evidence. Restraint score improved from 6% to 12%.

          

        


        

          
Tech Stack

          

            

              Base Model
              Qwen 2.5 7B Instruct
            

            

              Fine-tuning
              QLoRA (4-bit, rank 64)
            

            

              RL Algorithm
              GRPO (Group Relative Policy Optimization)
            

            

              Framework
              TRL + Hugging Face Transformers
            

            

              Tools
              17 connectors (GitHub, Drive, Gmail)
            

            

              Eval Suite
              50 tasks across 8 categories
            

          

        


        

          
Training Recipe

          

            

              01
              

                Define tool schemas
                
17 typed tool schemas across GitHub, Google Drive, and Gmail connectors with parameter validation.

              

            

            

              02
              

                Build eval suite
                
50 evaluation tasks spanning incident triage, spec audits, restraint tests, and multi-step workflows.

              

            

            

              03
              

                SFT for format
                
Supervised fine-tuning on curated traces to teach the think/tool_call/observation/answer format.

              

            

            

              04
              

                GRPO for judgment
                
Reinforcement learning with a composite reward: tool accuracy (30%), answer quality (30%), restraint (15%), efficiency (15%), ordering (10%).

              

            

            

              05
              

                Iterate
                
Re-evaluate, identify failure modes, generate targeted training data, and repeat.
